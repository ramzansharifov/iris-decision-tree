# Iris Flower Classification Using Decision Tree

## Project Description

This project is a simple machine learning study based on the Iris Dataset.  
The goal of the project is to classify iris flowers into three species using the Decision Tree algorithm.

The project was prepared as a homework assignment for the course Artificial Intelligence Systems.

## Dataset

The project uses the classic Iris Dataset.

The dataset contains 150 samples of iris flowers.  
Each sample has four numerical features and one target class.

## Features

The model uses the following features:

- sepal length
- sepal width
- petal length
- petal width

## Target Classes

The model predicts one of the following iris species:

- setosa
- versicolor
- virginica

## Algorithm

The main algorithm used in this project is:

- Decision Tree Classifier

A Decision Tree is a machine learning algorithm that makes decisions by splitting the data according to feature values.  
Its logic is easy to understand because it is similar to a sequence of simple if-else rules.

## Project Steps

The project includes the following steps:

1. Loading the Iris dataset
2. Converting the dataset into a pandas DataFrame
3. Displaying basic information about the dataset
4. Splitting the dataset into training and testing sets
5. Training the Decision Tree model
6. Making predictions on the test data
7. Evaluating the model using accuracy, classification report and confusion matrix
8. Visualizing the Decision Tree
